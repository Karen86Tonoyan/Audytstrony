# Ollama Agent - Rozszerzenie przeglądarki

Rozszerzenie Chrome/Edge dla Ollama - pełnoprawny agent AI do programowania.

## Funkcje

- **Chat z AI** - rozmowa z lokalnym modelem Ollama
- **Generowanie kodu** - tworzenie backend i frontend
- **Tworzenie aplikacji** - kompletne projekty (React, Vue, Node, Python)
- **Context menu** - prawy klik na zaznaczony kod
- **Side Panel** - panel boczny jak Claude

## Instalacja

1. Otwórz `chrome://extensions/`
2. Włącz "Tryb dewelopera"
3. Kliknij "Załaduj rozpakowane"
4. Wybierz folder `browser-extension`

## Wymagania

- Ollama uruchomiona lokalnie (`ollama serve`)
- Model: `ollama pull llama3.2`

## Skróty

- `Ctrl+Shift+O` - otwórz popup
- `Ctrl+Shift+L` - panel boczny
